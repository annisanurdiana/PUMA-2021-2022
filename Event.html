Event Projects

01

Compsphere

This event is open for the general public is a work plan from Human Resource Development division. This event is the biggest computing event. The event itself consists of :Seminar, Workshop, and Competitions.

02

Compgath

This is a private event between freshmans with computing students and this is a work plan from Internal Relation division. This event usually consist of introductory programme to familiarize freshmans with university environment.

03

CSGO

This event is a work plan from Arts And Sports division. This is a competition event between computing students. The activities performed are: E-Sports, Sports, and Academics.

04

SOPU Prep

Sopu Prep(sport olympiad president university preparation) is a work plan from Arts And Sports division. This event is a preparation/training event pre-Sopu leading up to Sopu. The gist of this event is to prepare/train computizens who are going to compete in Sopu.

05

Sospro

Social project is a work plan from External Relation division. This is a social event between computizens to the community as a way to give back to them.

06

Temu Alumni

A work plan from Internal Relation division. This is an experience sharing event between computing alumni and computing students (Computizens), usually consists of seminar/workshop and experience sharing as well from alumnus of computing.

07

Compclub

This event is a collaboration event from Human Resource Development division with Research and Technology division. It is an event to provide knowledge and training to Computizens related to technology.

08

Compvist

A work plan from External Relation division. It is an event for students to gain knowledge, insight, and get to know more about work life in computer/technology field.

09

Compstudy

Comparative study is a work plan from External Relation division. It is an annual event where PUMA COMPUTING do a comparative study with another major association from a different university.

10

Comphealth

A workplan by Art and Sports division. An event that is specifically aimed for Computizens who want to do sports activities together such as swimming, futsal, volleyball, basketball, badminton, and e-Sports (via Discord).

11

Compreciation

Computing Appreciation is a work plan from Human Resource Development division where its committees give some appreciation to Computizens for their hard work in the academic and non-academic fields.

12

Compshadow

Computing Shadow is a work plan from Internal Relation division. It is a bonding event between PUMA COMPUTING members. It consists of two types of event, ‘Welcoming day’ and ‘Farewell Day
